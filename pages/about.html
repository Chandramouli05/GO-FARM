About Go-Farm

This farm can be managed by the combination of following three phases of methodoloy in this project , The first phase was farm management here we can manage the planting & harvesting time. The Second phase was Iot Monitoring , Implementation of Iot devices like Node MCU , Soil Moisture sensor & DHT11 sensor. In this the monitoring of the soil moisture , Air Humidity, Temperature can easily be monitored throught that and also Implemented other device like 1-channel relay module with Motor pump (mini DC 12V) through these we can irrigate water to the plants remotely which will be can save water utilization.The third phase was plant disease detection, mostly farmers werenâ€™t aware of infected plants in nowadays. In smartphone world we can take picture or upload picture of infected plant leaves, using TensorFlow we trained models using machine learning, which scan and remedial suggestions for the plant disease. These phases are integrated together in one application using Progressive Web application method we can able to use this application as web and also as Android APK in the device.
